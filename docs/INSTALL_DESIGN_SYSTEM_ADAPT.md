# Design System Adapt Install

## Files

- Skill folder: `design-system-adapt/`
- Zip package: `design-system-adapt-skill-package.zip`

## Install

1. Unzip `design-system-adapt-skill-package.zip`.
2. Copy the `design-system-adapt` folder into your local skills directory.
3. Restart the agent session or reload skills.

## Use

- Invoke with `$design-system-adapt`
- Best for projects that use one canonical design-system source and generate AI, Figma, or runtime adapters from it.

## Notes

- This package has no hardcoded dependency on any local absolute filesystem path.
- It works best in repositories that already have a canonical source plus publishable adapter outputs.
